# IntelliSenseLocalizer
a tool for generate Localized IntelliSense files. 用于生成本地化IntelliSense文件的工具。

## How to use

### 1. install the tool
```shell
dotnet tool install -g islocalizer --prerelease
```

### 2. build file
```shell
islocalizer build
```

### 3. install builded file
```shell
islocalizer install
```
